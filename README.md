# RGB-Ball-Tracker-on-Raspberry-Pi-Pico-2
Interfacing a Arducam Mini 2MP Plus with a Raspberry Pi Pico 2 to put eyes on a microcontroller! It is able to detect red, blue, and green balls and the distance as well as x and y coordinates (on the image capture). This sub-project is to be implemented with a larger rover. All of this operates using only about 180 kb of memory.
